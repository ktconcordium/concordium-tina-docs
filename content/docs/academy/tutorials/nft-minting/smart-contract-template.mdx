title: "Generate a smart contract with the Concordium VSCode extension"
---

(smart-contract-template)=


Concordium has a direct extension on VSCode that allows you to set up your projects easily within minutes. Use the search in VSCode to find it and click **Install** and then click **Enable**.

![visual studio code marketplace showing Concordium smart contracts extension](/../images/mid-vscode-ext.png)

![visual studio code searching for concordium commands](/../images/mid-vscode-ext2.png)

Select **Initialize a smart contract project** and click your folder to set up your project.

![visual studio code creating project in folder](/../images/mid-vscode-ext3.png)

![visual studio code configuring project](/../images/mid-vscode-ext4.png)

Now build it. Open the Command Palette again, and select **Build contract**. Note that this option will build your contract and embed your schema in the module.

![visual studio code searching for build contract command](/../images/mid-vscode-ext5.png)

![visual studio code extension showing build finished](/../images/mid-vscode-ext6.png)
