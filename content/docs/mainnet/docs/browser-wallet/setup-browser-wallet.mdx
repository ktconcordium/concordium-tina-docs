(setup-browser-wallet)=


The Concordium Wallet for Web is a digital wallet that enables you to create and manage your Concordium
[identities](/docs/mainnet/docs/resources/glossary#identity) and [accounts](/docs/mainnet/docs/resources/glossary#account) and to create transactions, such as sending CCD.

To learn more about identities and accounts, see [Identity](/docs/mainnet/docs/protocol/identity#reference-identity) and [accounts](/docs/mainnet/docs/protocol/manage-accounts#managing_accounts).

Read the following guide to learn how to set up the wallet.

<Callout
  body={<>
The Concordium Wallet for Web extension is not supported in any of the web browsers below when used on a mobile phone or tablet.
  </>}
  variant="info"
/>

## Recovery phrase setup

If you are creating a new wallet, you must set up a seed phrase. This is a 24 word phrase that stores your private keys, identities, and accounts. You must write down and confirm your seed phrase. It is important to keep this seed phrase in a safe location in case you need to recover your wallet on a new device.

1. When you click **Create**, the 24 word seed phrase is shown. Write it down and click **Continue**.

   ![screen with recovery phrase and continue option](/../images/browser-wallet/new/recovery.png)

![screen with option for creating account](/../images/browser-wallet/new/create_account.png)

2. Select the Identity to use for the account and click **Create account**.

## Basic navigation

### Main page

The main page shows your account information and a menu accessible from the top-right corner. The page layout includes:

1. An account selector in the top left that lets you switch between accounts, showing the current account's address and a dropdown arrow.

2. Your total CCD balance.

3. Four quick action buttons: **Receive**, **Send**, **Earn**, and **Activity**.

4. A list of your tokens with their balances.

5. Option to manage your token list at the bottom.

   ![menu at top of window with drop-down expanded to show options](/../images/browser-wallet/new/main_page.png)

### Menu

Clicking the menu button ![button for opening menu](/../images/browser-wallet/new/menu_button.png) in the top-right corner opens a page with additional features:

01. **Identities**: Manage your identities
02. **Accounts**: View and manage accounts
03. **Seed phrase**: Access your recovery phrase
04. **Passcode**: Security settings
05. **Web3 ID**: Web3 identity features
06. **Earn**: Staking and rewards
07. **Network**: Network settings
08. **About**: App information
09. **Restore**: Recovery options
10. **Old UI**: Previous interface version
11. **NFT**: NFT management

The menu can be closed by clicking the **X** in the top-right corner to return to the main page.

![screen with option for creating account](/../images/browser-wallet/new/menu_screen.png)

### View account activity

1. Select the account from the account selector in the top left of the main page.
2. Click ![button for viewing wallet activities](/../images/browser-wallet/new/activity_button.png)
3. For transaction details, click on any specific transaction in the list.

### Find and share your account address

1. Select the account from the account selector in the top left of the main page.

2. Click ![button for viewing qr code](/../images/browser-wallet/new/receive_button.png)

3. Share your address:

   - You can let someone scan your QR code if they have the Concordium Legacy Wallet installed.
   - You can click **Copy Address** to copy the address and then paste it wherever needed.

(switch-network)=

### Change network

You can use the wallet on both Mainnet and Testnet without the need for a separate application. Use the **Network Settings** to switch between Mainnet and Testnet.

<Callout
  body={<>
Accounts and identities are NOT shared between mainnet and testnet.
  </>}
  variant="info"
/>


1. Click ![button for opening menu](/../images/browser-wallet/new/menu_button.png) to open the menu page.
2. Click **Network**.
3. Select the Network you want to connect to: Mainnet or Testnet. Note that the color of the user interface items in Mainnet is blue and the color of the user interface items in Testnet is green.

### Switch between old and new UI

You can switch between the old and the new wallet interface.

To access the old interface:

1. Click ![button for opening menu](/../images/browser-wallet/new/menu_button.png) to open the menu page.
2. Click **Old UI**.

To return to the new interface:

1. Go to Wallet settings in the old interface.
2. Click **Wallet X**.
