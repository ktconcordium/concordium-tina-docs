---
title: "Apply a transaction log filter in the Desktop Wallet"
---

(transaction-log)=


You can specify which transactions to show in the transaction log.

1. Go to **Accounts** and select the account that you want to apply the transaction log filter to.

2. Click **More options**.

3. Select **Transaction log**.

4. Select **Filters** in the upper right corner.

   > ```{image} ../images/desktop-wallet/dw-transaction-log-filter.png
   > ```

5. Specify the date range that you want the log filter to cover.

6. Select the transaction types you want displayed in the log.

7. Select **Search** if you want to apply the changes temporarily to look up specific transactions. Select **Save filters** if you want to save the filter until you apply new ones. Select **Clear** if you want to completely remove a filter.
