(export-transaction-logs)=


It is possible to export tansaction logs with [CCD Scan](/docs/mainnet/tools/ccd-scan/ccd-scan#ccd-scan) from the Concordium Wallet.

<accordion
  heading="Concordium Wallet"
  docText={<>
1. Go to the Your Accounts screen (tap the account identifier on the main screen).

2. Open Account Settings by tapping the gear icon in the upper right og the screen.

3. Tap **Export transaction logs**.

4. Tap **Save transaction logs** and browse to the saving location.
