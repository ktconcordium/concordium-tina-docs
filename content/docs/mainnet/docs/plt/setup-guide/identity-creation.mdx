title: "Create a DevNet identity"

(plt-identity-creation)=


This guide explains how to to create an [identity](/docs/mainnet/docs/protocol/identity#reference-identity) on DevNet. An identity is a prerequisite for creating accounts and working with Protocol-Level Tokens.

After [connecting to DevNet](/docs/mainnet/docs/plt/setup-guide/devnet-connection#plt-devnet-connection), follow these steps to create your identity:

1. In the wallet, go to **Identities**

   ![select identitties in the menu](/../Images/create-identity1.png)

![start creating new identity](/../Images/create-identity2.png)

3. Choose **identity provider**: Select **Generated IP 0**

   ![selecting-identity-provider](/../Images/request-identity.png)

![completing identity verification](/../Images/identity-verification.png)

5. Click **Done** to complete the identity registration process

   ![finishing identity request](/../Images/finish-identity-request.png)

6. Return to the wallet when finished.

You have now successfully created your identity on DevNet. You can use this identity to create accounts and start working with Protocol-Level Tokens (PLTs).
